Guia de HTML

La cabecera

La cabecera es el contenido del elemento <head>, que a diferencia del contenido del elemento <body>, no se muestra en la página.
Su función es contener los metadatos del documento.

      <head>
         <title>Mi primer HTML</title>
         <meta charset="utf-8">
         <meta name="author" content="Natalia">
         <link rel="stylesheet" href="MyEstilo.css">
      <head>

Título

Representa el título del documento.

Meta

Representa varios metadatos que no se pueden describir con las otras etiquetas. En este ejemplo se ha usado para indicar codificación del documento y el autor.

Link

Permite introducir documentos externos. Debe tener el atributo rel.

Style

Permite añadir estilo al documento mediante el uso de CSS.
