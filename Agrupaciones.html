Agrupaciones

Párrafo <p>

Representa un párrafo.

<p>Esto es un párrafo.</p>
<p>Esto es otro párrafo.</p>

Texto preformateado <pre>

El texto se representa tal y como se ha escrito, a diferencia del párrafo, que obvia cosas como saltos de página y tabulaciones.

          <pre>
                Este texto se verá
            exactamente así
                      cuando aparezca en el navegador.
          </pre>

Listas

Hay 3 tipos de listas: listas ordenadas, listas desordenadas y listas de definiciones.

Listas ordenadas <ol>

Cada elemento de la lista usará el elemento <li>.
El elemento <ol> puede tener varios atributos:

Reversed
Indica que la lista es descendente.
Start
Indica el número por el que la lista empieza.
Type
Indica el tipo de marcador que se va a usar. En caso de que no aparezca, se usarán números.

Listas desordenadas <ul>

Como en las listas ordenadas, cada elemento de la lista usa el elemento <li>.

Listas de definiciones <dl>

Se usan para listas en las que cada elemento tiene un nombre y un valor asociado, como definiciones, descripciones, etc. Se usa el elmento <dt> para definir el término y <dd> para la descripción.
Por ejemplo: 

          <dl>
            <dt> Director
            <dd> George Lucas
            <dt> Actores
            <dd> Mark Hamill
            <dd> Harrison Ford
            <dd> Carrie Fisher
          </dl>

Tablas <table>

Elementos

<thead>
Encabezado de la tabla.
<tbody>
Cuerpo de la tabla.
<tfoot>
Sumario de la tabla.
<caption>
Leyenda de la tabla.
<tr>
Añade una fila
<td>
Añade una columna

Tabla
      Fila
        Columna
        Columna
        ...
      Fila
        Columna
        ...

<table>
      <tr> Table row
        <td> Table data
        <td> Table data
        ...
      <tr> Table row
        <td> Table data
        ...

Atributos

border
TABLE. Indica el tipo de borde de la tabla. Si no está, la tabla no tendrá borde.
rowspan
TD. Expande una columna.
colspan
TD. Expande una fila.

Otros elementos

Citas <blockquote>

Se usa para representar citas de otras fuentes.

Línea horizontal <hl>

Crea una línea horizontal.

<div>

Este elemento se usa como comodín. No tiene ningún significado especial, pero se suele usar para ayudar a dar estilo.
